# Quickstart Guide

## Prerequisites

- Node.js (v16 or higher)
- npm or yarn
- React Native CLI
- Android Studio (for Android development)
- Xcode (for iOS development)

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/giovaniguerra/learn-redux.git
   cd learn-redux
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

4. Run the app:
   - For Android:
     ```bash
     npm run android
     ```
   - For iOS:
     ```bash
     npm run ios
     ```

## Testing

- Run unit tests:
  ```bash
  npm test
  ```

## Contribution

- Follow the Git Flow branching model.
- Ensure all commits follow the Conventional Commits specification.
- Document changes in the `changelogs.md` file.
